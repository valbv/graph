========================================================================
	Инструкция по пользованию программой
========================================================================
Программа расчитывает оптимальный маршрут в заданном графе.
Используется алгоритм Дейкстры.
Входные данные задаются в файле input.txt (расположен в папке проекта).


========================================================================
    Как задать свой граф, точку отправления и точку назначения:
========================================================================
1) Открыть файл input.txt

2) Перечислить вершины графа (имена вершин) после строки
"[Nodes]"
Имя каждой вершины указывается на отдельной строке

3) Перечислить дуги (рёбра) графа после строки
"[Arcs] (from to weight)"

Данные дуги указываются через пробел:
	- имя вершины из которой выходит дуга
	- имя вершины в которую входит дуга
	- вес дуги

Данные каждой дуги указываются на отдельной строке. 

4) Задать имя вершины из которой нужно начать движение после строки
"[Path-begin]"

5) Задать имя вершины в которую нужно попасть после строки
"[Path-end]"

6) Сохранить файл input.txt и запустить программу
========================================================================
	Результат работы программы:
========================================================================
В консоли будет отображён оптимальный путь перемещения из начальной точки
в конечную
